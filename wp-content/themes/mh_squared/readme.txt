==================================
Description
==================================

MH Squared is a responsive magazine WordPress theme that comes with helpful options and powerful widgets just waiting for your contents. This magazine theme gives you the ability to create an awesome-looking online magazine using a modern and sleek layout that is fully-customizable using the advanced theme customizer and an array of custom widgets.
